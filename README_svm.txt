README
------
1) To run, simple execute applySVM.m in Matlab. 

libsvm 3.18 needs to be installed prior to running. MEX files are provided for windows. Must be compiled from source for other operatng systems. See http://www.csie.ntu.edu.tw/~cjlin/libsvm for details. 

2) Uncomment the indicated lines in applySVM.m to retrain the model again if necessary.
