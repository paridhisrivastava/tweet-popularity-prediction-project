The following command chain is required to run the classifier for Random Forest Classification:
1. Install csv, numpy, sklearn, pandas, pandas_confusion, matplotlib packages in python
2. Run randomforestclassifier_retweet.py to get the prediction, plot containing the information regarding the importance of the features in prediction, confusion matrix and the accuracy, based on the labelling done by the number of retweets. 
3. Run randomforestclassifier_fav.py to get the prediction, plot containing the information regarding the importance of the features in prediction, confusion matrix and the accuracy, based on the labelling done by the number of favorites.
